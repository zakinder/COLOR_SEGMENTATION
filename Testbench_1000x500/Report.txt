RGB HSV2RGB Match-Input-Color 1000x500 Testbench Report
=======================================================

Pipeline:
  RGB BMP input -> rgb_hsv -> hsv2rgb -> RGB BMP output

Default settings:
  PREFILTER_MODE = 0
  ENABLE_FLAT_COLOR = false
  LOW_SATURATION_TO_GRAY = false
  LOW_VALUE_TO_BLACK = false

Purpose:
  Validate that the HSV round-trip path preserves the original input image colors
  instead of producing a flattened/posterized image.

Input:
  Input_Image/1000_500.bmp

Output:
  Generated_Outputs/output_rgb_hsv2rgb_match_input.bmp

BMP support:
  24-bit BGR BI_RGB only for this 1000x500 package
